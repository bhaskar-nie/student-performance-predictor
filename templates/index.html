Welcome to the Student Performance Prediction Tool

This project demonstrates the application of machine learning to predict student performance based on various factors such as gender, learning support, parental education, and test preparation completion.

Project Overview

This project is a part of my Minervaa Assignment Task, which aims to showcase the predictive power of machine learning models in educational contexts. The task involves building a machine learning model to predict a student's future academic performance based on their past marks and other factors.

By analyzing a dataset of students' demographics and performance data, the model predicts the student's performance in the upcoming test. The insights from the predictions can help identify factors influencing academic outcomes and provide valuable information for educational stakeholders.

Project Features

  • Prediction of student performance in the upcoming test based on various demographic and academic factors.
  • Analysis of factors like gender, learning support, parental education, and course completion on student performance.
  • A simple, user-friendly web interface for inputting student data and receiving performance predictions.

Model Training and Evaluation

The machine learning model used in this project is trained using various techniques, including Linear Regression, Random Forest Regressor, K-Neighbors Regressor, and Decision Tree. The dataset is split into training and testing sets to evaluate the model's performance. The model predicts student's upcoming test scores based on the features provided.

Model Performance

Below are the models evaluated for predicting student performance along with their R² scores (which indicates the accuracy of each model):

Model Name R² Score
Linear Regression 0.937908
Random Forest Regressor 0.917223
K-Neighbors Regressor 0.880506
Decision Tree 0.866690

Model Accuracy

The best model in this project was Linear Regression, achieving an accuracy of 93.7%. Other models like Random Forest Regressor and K-Neighbors Regressor also performed well, while the Decision Tree model had a relatively lower accuracy.
